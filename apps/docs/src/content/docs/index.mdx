## Quick Example

```typescript
import { Effect, Schema } from "effect"
import { api, auth } from "@effect-kit/core"

// Define your schema with Effect Schema
const Todo = Schema.Struct({
  id: Schema.String,
  title: Schema.String,
  completed: Schema.Boolean,
})

// Create type-safe API routes
export const todoApi = api.group("todos", {
  middleware: [auth.required()],
  
  routes: {
    create: api.post("/")
      .schema(Todo)
      .handle(({ body, user }) =>
        Effect.gen(function* () {
          const db = yield* Database
          return yield* db.insert("todos", {
            ...body,
            userId: user.id
          })
        })
      ),
  }
})
```

## Get Started in Seconds

```bash
npx create-effect-app@latest my-app
cd my-app
bun dev
```

Your app will be running at [http://localhost:3000](http://localhost:3000) 🎉
